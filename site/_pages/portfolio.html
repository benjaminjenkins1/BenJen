Here are a few of the things I've made.


Applause - Python, PostgreSQL

A clap-based analytics and approval tracking tool. Create an account and add a script to your articles that will track page views and top referrers on your site, and add a button that allows your users to “clap” for the article. View reports on the data generated in your daashboard, and get a better understanding of which of your articles people like most, as well as where your traffic is coming from.

BenOS - 6502 Emulator and OS - TypeScript

Browser-based emulator and operating system using an instruction set based on that of the 6502 microprocessor. Features include a console, shell, filesystem, and more!

Magic Frame - Python, SQLite

My attempt at a smart photo frame system. Users create accounts with just a phone number (SMS integration with Twilio!), and upload photos that are displayed on a frame in their home and in frames owned by users they are friends with. The frame application is designed to run on a Raspberry Pi, and runs a discovery server and the photo display. A user uses the web application to discover and claim a frame on their network, to add friends, and upload photos.

Marist Writing Advisor - PHP, Javascript, Python

The Marist Writing Advisor is a web application designed to help students improve the quality of thier writing. It uses a sentence processing system designed by my professor at Marist, Pablo Rivas, to evaluate the quality of sentences based on a set of rules developed by the English department at Marist. It also uses the Watson Natural Language Understanding API’s to provide ancillary feedback on a student’s writing, including information on the language, tone, and personality of their writing. After developing the prototype interface for this application with IBM Joint Study, I continued work on this system with a team of other students as my Capstone project. You’ll need a Marist account to access this tool.

MyMovieList API - Golang, PostgreSQL

A JSON API designed for keeping track of a list of movies you want to watch. Written in Go, this API integrates Google’s and Facebook’s OAuth2 authentication services and uses Redis as its session store. I had planned to build a mobile app to consume this API, but that’s been put on hold for now. I might revisit it in the future, though.

Limbo Lost & Found - PHP, MySQL

One of my first software development projects which can be described as a complete system. This web application allows users to create an account, then log in and provide information about items they have either lost or found on the Marist campus. It also allows users to claim an item and provides a means to get in touch with the person who found it.
